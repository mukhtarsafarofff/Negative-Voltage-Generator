(ENG) In this circuit can be made more efficient as well as increasing the output voltage by using Schottky diodes instead of generic silicon. This is because Schottky diodes  results in a larger negative voltage..

(DEU) In dieser Schaltung kann durch die Verwendung von Schottky-Dioden anstelle von generischem Silizium.Auf diese weise erzielt  eine höhere Effizienz  und die Ausgangsspannung erhöht werden.  Dies liegt daran, dass Schottky-Dioden zu einer größeren negativen Spannung führen.
